## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Customization

Edit the content in the component files:
- `components/Hero.tsx` - Hero section with your name and tagline
- `components/Experience.tsx` - Your work experience
- `components/Skills.tsx` - Your skills and technologies
- `components/Projects.tsx` - Your projects
- `components/Contact.tsx` - Contact information and links
